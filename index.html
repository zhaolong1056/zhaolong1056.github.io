Python线程

发表于 2019-10-10 分类于 Python

线程 thread

  • 是操作系统能够进行运算调度的最小单位。

  • 它被包含在进程之中，是进程中的实际运作单位。

  • 一条线程指的是进程中一个单一顺序的控制流，一个进程中可以并发多个线程，每条线程并行执行不同的任务。

  • 在Unix System V及SunOS中也被称为轻量进程（lightweight processes），但轻量进程更多指内核线程（kernel thread），而把用户线程（user thread）称为线程。
